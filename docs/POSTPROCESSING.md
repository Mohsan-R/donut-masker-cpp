# Post-Processing

## Overview

Initial segmentation provides a good approximation of the donut regions, but the raw masks often contain small imperfections caused by thresholding, lighting variations, and image noise.

The purpose of the post-processing stage is to refine these masks by removing artifacts, smoothing boundaries, and improving geometric consistency while preserving the true donut shape.

The complete refinement pipeline is illustrated below.

```
Initial Donut Mask
        │
        ▼
Morphological Opening
        │
        ▼
Morphological Closing
        │
        ▼
Median Filtering
        │
        ▼
Contour Approximation
        │
        ▼
Refined Donut Mask
```

---

# Why Post-Processing?

Even after geometric filtering, binary masks may contain:

- Small isolated pixels
- Jagged boundaries
- Tiny holes
- Broken contour edges
- Staircase artifacts introduced by thresholding

Without refinement, these imperfections reduce segmentation quality and produce less accurate overlays.

---

# 1. Morphological Opening

The first refinement step applies a morphological opening operation.

Opening consists of:

```
Erosion
↓

Dilation
```

## Purpose

Opening removes:

- isolated foreground noise
- tiny connected components
- edge artifacts
- small binary defects

Because erosion removes small objects before dilation restores the main shape, legitimate donuts remain intact while small artifacts disappear.

---

# 2. Morphological Closing

After opening, a closing operation is applied.

Closing consists of:

```
Dilation
↓

Erosion
```

## Purpose

Closing repairs:

- small gaps
- broken edges
- fragmented contours
- tiny cracks

This operation improves contour continuity and produces more complete donut boundaries.

---

# 3. Median Filtering

Although morphology removes most noise, small binary irregularities may still remain.

A median filter is therefore applied to the binary mask.

## Benefits

Median filtering:

- preserves object edges
- removes isolated pixels
- smooths contour boundaries
- suppresses impulse noise

Unlike Gaussian smoothing, the median filter maintains sharp binary transitions.

---

# 4. Contour Extraction

After refinement, contours are extracted again from the cleaned binary mask.

Each contour represents a candidate donut boundary.

Using refined contours instead of the original segmentation improves both visual quality and geometric stability.

---

# 5. Contour Approximation

The extracted contours are simplified using polygonal approximation.

OpenCV's contour approximation algorithm reduces unnecessary contour points while preserving the overall donut geometry.

## Advantages

- smoother boundaries
- fewer jagged edges
- reduced contour complexity
- cleaner visualization

The resulting contours are better suited for industrial inspection and visualization.

---

# Hole Refinement

The donut hole undergoes a similar refinement process.

Candidate hole regions are filtered using geometric constraints including:

- minimum area
- maximum area
- circularity

Only contours satisfying all validation criteria are retained.

This prevents noise inside the donut from being incorrectly identified as holes.

---

# Final Mask Generation

The final donut mask is generated by subtracting the validated hole mask from the refined donut mask.

```
Refined Donut Mask
          │
          ▼
Subtract Hole Mask
          │
          ▼
Final Donut Mask
```

This produces a binary mask representing only the edible donut region.

---

# Overlay Generation

To visually verify segmentation quality, refined contours are drawn on the original image.

Color convention:

| Color | Meaning |
|--------|---------|
| Green | Outer donut boundary |
| Red | Donut hole |

The overlay provides an intuitive comparison between the detected masks and the original image.

# Effect on Segmentation Quality

The refinement stage improves several aspects of the final masks.

| Metric | Before | After |
|---------|--------|-------|
| Noise | Moderate | Low |
| Boundary Smoothness | Fair | High |
| Hole Consistency | Moderate | High |
| Overlay Accuracy | Good | Excellent |

These improvements increase both visual quality and robustness without altering the fundamental donut geometry.

---

# Computational Cost

The post-processing operations consist entirely of classical image processing techniques.

Operations include:

- morphology
- median filtering
- contour extraction
- contour approximation

These operations are computationally inexpensive and execute efficiently, making the pipeline suitable for batch processing of industrial datasets.

---

# Summary

The post-processing stage transforms the initial binary segmentation into clean, well-defined masks suitable for industrial inspection.

Through a combination of morphological operations, median filtering, contour refinement, and geometric validation, the pipeline significantly improves boundary quality while maintaining computational efficiency and reproducibility.
